
# AMAL Prototype

## Deployment Files
- `amal_app.py`
- `utils.py`
- `requirements.txt`
- `logo.png`

## Deployment Steps
1. Ensure these files are in the root of your GitHub repo.
2. Commit and push to the `main` branch.
3. On Streamlit Cloud, set **Main file path** to `amal_app.py`.
4. Click **Deploy**.
