# AMAL Prototype

## Deployment Files
- `amal_app.py`
- `utils.py`
- `requirements.txt`
- `logo.png`
- `README.md`

## Deployment Steps
1. Delete all other files from your repo.
2. Copy these five into the **root** of the repo.
3. Commit & push to `main`.
4. On Streamlit Cloud, set **Main file path** to `amal_app.py`.
5. Click **Deploy** and open the provided URL.
